# 🏢 HR Analytics Dashboard

![Dashboard](q10_hr_dashboard.png)

## 📌 Project Overview
Performed complete HR data analysis on 933 employees
to uncover insights about salary structure, gender equity,
team performance, bonus distribution, and senior management
compensation patterns.

## 🛠️ Tools Used
- Python 3
- Pandas — data cleaning and analysis
- Matplotlib — charts and dashboard
- Seaborn — statistical visualizations

## 📊 Business Questions & Key Findings

| # | Question | Finding |
|---|----------|---------|
| Q1 | Headcount per team? | Client Services is largest |
| Q2 | Salary distribution? | Uniform Rs 35K–150K, mean Rs 90K |
| Q3 | Highest paying team? | Engineering — Rs 94,582 avg |
| Q4 | Gender pay gap? | Only 1.25% — no significant gap |
| Q5 | Senior Management %? | 50.2% — unusually high |

## 💡 Key Insights
- ✅ No gender pay gap — fair compensation structure
- ✅ Females lead top 10 highest earners
- ⚠️ Unassigned team has highest bonus — needs investigation
- ⚠️ Business Development pays Non-Senior MORE than Senior
- ⚠️ Seniority carries only 1.4% salary premium
